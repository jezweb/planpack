# Coverage attestation — Acacia House (Adelaide)

This is a **Preliminary-stage architectural set** (10 sheets, one issuer, June 2023). It is a complete architectural design set but it is *not* a construction documentation package. This page states, family by family, what was extracted and what this set simply does not carry — a silent gap reads as "covered" to the next agent, so absences are named explicitly.

Extraction method note: the PDF's text layer carries **title blocks only**; every drawing annotation is outlined vector. All extraction was vision-on-render (150 dpi sheet renders + 300–600 dpi crops). Deterministic script counting of text tags was not possible on this set; dimension-chain closure checks and one vector-layer geometry read (site block rectangle) were used as deterministic cross-checks.

## Design criteria (hot fields)

**None stated anywhere in this set** — no wind class, soil class, site class, BAL, exposure, concrete grades, or geotech reference. Expected: this is a Preliminary generic design with no structural engineering and no site. Any quote built on this pack must carry these as exclusions/TBC.

## Trade views

- **Builder/estimator (residential)**: can produce a preliminary joinery takeoff (complete), GFA-based budget figures (stated), and a fabric scope with TBC materials. Cannot price structure, roof area, services, or finishes. Site costs entirely TBC (indicative block).
- **Electrical**: derivable = 5 ceiling fans (approximate). Committed-but-undrawn = everything else; no BASIX/NCC commitment schedule is present either. Verdict: **not quotable from this set** — flag as by-others/tbc.
- **Glazing/joinery**: fully quotable *as nominal* — 15 units with dims and operations; frame sections notional, laminated-glass locations to be confirmed by fabricator, U/SHGC in the absent specifications document.
- **Concreter, roofer, plumber, HVAC**: scope indicated but undesigned — allowance/tbc only.
